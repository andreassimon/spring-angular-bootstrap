Spring / Angular Bootstrap
=========

Provide a starting point for Spring / Angular applications with some goodies:
 - Gradle
 - Spring Boot
 - Spring Web
 - Spring Security
 - Spring Actuator
 - Spring Loaded
 - Swagger
 - Lizenzliste
 - Asciidoctor
 - JUnit 5
 - Gatling
 - (Groovy)
 - (Cucumber mit Groovy-Bindings)
